# 📚 Taller – Estructuras de Datos en JavaScript

## Descripción

Este proyecto fue desarrollado como evidencia de aprendizaje para practicar el uso de las principales estructuras de datos de JavaScript.

La aplicación está dividida en cuatro módulos independientes que permiten comprender el funcionamiento de **Array**, **Set**, **Map** y la transformación entre estas estructuras.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

## Funcionalidades

### 📦 Aplicación 1 – Array

- Crear un Array.
- Agregar elementos al inicio y al final.
- Eliminar elementos.
- Mostrar el contenido del Array.

### 📚 Aplicación 2 – Set

- Crear un Set.
- Agregar libros.
- Eliminar libros.
- Mostrar elementos sin duplicados.

### 🗂️ Aplicación 3 – Map

- Crear un Map con meses del año.
- Buscar valores por clave.
- Agregar nuevos registros.
- Mostrar el contenido completo.

### 🔄 Aplicación 4 – Transformaciones

- Convertir un Array en Set.
- Convertir un Set en Map.
- Visualizar el resultado de cada transformación.

---

## Objetivo del proyecto

Aplicar los conceptos fundamentales de las estructuras de datos en JavaScript mediante una interfaz web sencilla, organizada y fácil de usar.
